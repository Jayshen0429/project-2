Web 智能水族箱



智能水族箱擁有更加智能化的監控和管理系統,能夠實時監測、溫度、PH值等關鍵參數。 將物聯網(IoT)技術應用於水族箱管理，我們將帶來多重意義和影響。 智能水族箱的推出將大幅提升水族飼養的效率和品質。 此外，智能水族箱的推廣還將促進物聯網(IoT)技術在家居生活中的應用和普及。

About Project

Django

基於 Django 框架構建 MVC 架構，包括模型（Model）、視圖（View）和控制器（Controller）的設計和開發。

Firmware

使用 Raspberry Pi 作為核心平台，搭建 Sensor 體系。透過 Django MVC 架構，我們打造出了一個Web 應用界面。

HttpRequest

發送一個 HTTP 請求到Respberry。
控制 Respberry 上的硬體設備（如餵食器、水泵、LED 等）。

SQL

通過 MySQL 儲存資料，使用者可以查看目前水族箱溫度及 PH 質。以及24小時的曲線變化!

Webcam

Mjpg-streamer 是用來將視訊網路串流化的工具，透過其內建的 MJPG 解碼器，可以提供良好的影像擷取速率和畫質表現。

Cart

擁有 Web 購物車介面，讓消費者與水中生物做朋友也能同是享受購物樂趣!

Web Features

即時影像處理技術讓您無論身在何處，都能觀察每一條魚的動態和每一株水草的變化，彷彿身臨其境。

即時顯示水族箱的溫度和 pH 值，不論是白天還是夜晚，您都能即時了解水質狀況，為您的水生朋友提供最貼心的照顧。

點選按鈕可以控制LED燈、餵食器、抽水，享受與寵物互動的樂趣!

可查看近期10次的控制資料及24小時內PH與溫度變化!
